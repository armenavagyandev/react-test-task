# React Test Task

This is a React project built with Vite and TypeScript. The project includes reusable UI components, table and pagination functionality, custom hooks, and state management using Effector.
## Installation

Make sure you have Node.js v24.1.0 installed. Then install dependencies using yarn

```bash
yarn install
```

Run the project

```bash
yarn dev
```


## Features
- Reusable UI components
- Table with dynamic data handling
- Pagination and sorting functionality
- Global search for table data
- Custom React hooks for common logic
- State management with Effector
- TypeScript support for type safety
